# ashrain.out

기하학의 아름다움을 만나다 — 학원 수학 교육 웹앱

## 개발 환경 셋업

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 배포 (Vercel)

1. GitHub에 이 프로젝트를 push
2. vercel.com에서 GitHub 연결
3. 자동 배포 완료

## 기술 스택

- React + Vite
- SVG 기반 인터랙티브 기하학
- Firebase Firestore (Auth + DB)
- Vercel (호스팅)
